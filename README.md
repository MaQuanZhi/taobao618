
# 安卓 淘宝618列车 喵币自动领取脚本 需配合auto.js使用
# auto.js下载链接
(https://share.weiyun.com/tMNhozS2 密码:rvfn6m)

# 淘宝618列车 自动版
1. 打开auto.js
2. 开启无障碍模式
3. 在aotu.js中添加automatic.js文件
4. 运行脚本automatic.js

# 京东618叠蛋糕 自动版
1. 打开auto.js
2. 开启无障碍模式
3. 在aotu.js中添加jd618.js文件
4. 运行脚本jd618.js

# 淘宝618列车 手动版
1. 打开auto.js
2. 开启无障碍模式，开启悬浮窗
3. 在aotu.js中添加manual.js文件
4. 淘宝打开喵币领取界面
5. 通过悬浮窗运行脚本manual.js
